Help

How to draw Line
  1. Select Line in shape
  2. Click a spot in Canvas to make the first dot
  3. Drag on canvas
How to draw Square
  1. Select Square in shape
  2. Click a spot in Canvas to make the first dot
  3. Drag on canvas
How to draw Polygon
  1. Select Polygon in shape
  2. Click a spot in Canvas to make the dots of the polygon
  3. Click on the first dot you make to complete the polygon
How to edit length of shape
  1. Select a shape from "All shapes"
  2. Change the length of the shape
How to edit shape
  1. Select edit on the menu
  2. Drag a dot of the shape you want to edit
How to clear the canvas
  1. Click clear on the menu section
How to edit shape
  1. Select edit on the menu
  2. Drag a dot of the shape you want to edit
How to change shape color
  1. Select shape you want to change the color in "All Shapes"
  2. Click color section and edit the color with Color Wheel
How to save drawing
  1. Write name of the file you want your drawing to be saved in
  2. Click "Donwload save file" on Save and Load section
How to load drawing
  1. Click "choose file" on Save and Load section
  2. Browse the file on your File Manager (File must be a .json type)
  3. Click "open"
